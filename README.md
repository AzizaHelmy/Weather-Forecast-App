# Weather-Forecast-App
an android app that displays the weather status and the temperature from  your location
# Features:
 -MVVM

 -Navigation Components

 -View binding
 
 -WorkManager
 
 -coroutines
 
 -Romm DB
 
 -Retrofit
 
 -Live Data

 -KTX
